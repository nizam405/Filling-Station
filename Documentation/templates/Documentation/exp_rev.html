# খরচের খাত

খরচের খাত যুক্ত করার জন্য সাইডবার থেকে “খরচের খাত” নির্বাচন করি। এখানে নাম ও ক্রম লিখে বাটনে ক্লিক করি। একই পেজে খরচের খাতের তালিকা দেখা যাবে।
